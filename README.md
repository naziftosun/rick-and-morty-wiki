# Rick and Morty Wiki

A React-based wiki application for exploring the Rick and Morty universe, featuring characters, episodes, and locations.

## Features

- Browse characters, episodes, and locations
- Search functionality for all sections
- Detailed views for each item
- Responsive design with Tailwind CSS
- Load more pagination

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- React
- React Router
- Tailwind CSS
- Vite
- Rick and Morty API
